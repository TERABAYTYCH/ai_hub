# 🚀 Microservices Hub Platform

Добро пожаловать в монорепозиторий модульной платформы. Проект построен на микросервисной архитектуре с использованием NestJS, React (Vite + Module Federation), MySQL, TypeORM и Kafka.

## 📂 Навигация по проекту (Где что искать)

Вся кодовая база разделена на независимые приложения и общие библиотеки с помощью Yarn Workspaces и Turborepo.

| Директория      | Описание                   | Что внутри                                                                                |
| --------------- | -------------------------- | ----------------------------------------------------------------------------------------- |
| `docs/`         | **Системная документация** | Правила для AI, архитектурная карта, пайплайн работы и ADR. **Читать в первую очередь!**  |
| `apps/hub/`     | **Ядро (Hub)**             | `frontend/` (управление лицензиями), `backend/` (Auth, JWT, мастер-база устройств).       |
| `apps/pulse/`   | **Мониторинг (Pulse)**     | `frontend/` (графики), `backend/` (сбор метрик).                                          |
| `apps/service/` | **Обслуживание (Service)** | `frontend/` (отчеты), `backend/` (задачи на обслуживание).                                |
| `apps/control/` | **Управление (Control)**   | `frontend/` (панель управления), `backend/` (отправка команд).                            |
| `libs/`         | **Общий код**              | `contracts/` (DTO, типы), `ui-kit/` (React-компоненты), `core-backend/` (утилиты NestJS). |
| `nginx/`        | **API Gateway**            | Конфигурация Nginx для маршрутизации запросов к микросервисам.                            |

## 🛠 Инфраструктура

В корне проекта находится `docker-compose.yml`. Он описывает всю локальную инфраструктуру:

- Базы данных MySQL для каждого сервиса.
- Брокер сообщений Kafka (и Zookeeper).
- Nginx (API Gateway).
- Контейнеры самих бэкендов и фронтендов (с hot-reload для разработки).
