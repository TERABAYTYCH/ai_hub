## Что сделано

Реализовано в рамках тикета 002.1.

`useDynamicRoutesConfig` принимает `microservicesAccess` параметром и для каждого manifest:
- Проверяет `isLocked = microservicesAccess[manifest.serviceId] === false`
- При `isLocked`:
  - Добавляет route `/${serviceId}/lock` с `<ProtectedRoute><Layout><LockPage /></Layout></ProtectedRoute>`
  - Добавляет wildcard `/${serviceId}/*` с Navigate на `/${serviceId}/lock`

**Примечание:** Service (`microservices.service`) НЕ проверяется — проверяются только другие сервисы (Pulse).
