## Что сделано (в рамках 002.1)

`useDynamicRoutesConfig` принимает `microservicesAccess` параметром и:
- Проверяет `isLocked = microservicesAccess[manifest.serviceId] === false`
- Для заблокированных сервисов добавляет route `/${serviceId}/lock` с LockPage
- Добавляет wildcard `/${serviceId}/*` с редиректом

**Примечание:** Service не проверяет сам себя (`microservices.service`). Проверяются только ДРУГИЕ сервисы (Pulse).
