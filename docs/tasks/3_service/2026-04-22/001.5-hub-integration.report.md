## Как это работает

Hub динамически загружает `manifest.json` из каждого сервиса:
- При загрузке Hub, `useMicroserviceManifests` делает fetch к `http://service.lvh.me/assets/manifest.json`
- manifest.json содержит `serviceId: "service"` и `navigation` (Dashboard, Devices, Settings)
- Sidebar отображает Service в меню
