## Что сделано

Создан frontend для микросервиса Service путём копирования из Pulse и адаптации.

### 1. Копирование

Скопирована директория `apps/pulse/frontend/` → `apps/service/frontend/`

### 2. Изменённые файлы

| Файл | Изменение |
|------|-----------|
| `package.json` | name: `@ject-hub/service-frontend` |
| `vite.config.ts` | name: `'service'`, exposes: Dashboard, Devices, Settings |
| `.env` | VITE_API_URL=http://api.service.lvh.me |
| `.env.example` | VITE_API_URL=http://api.service.lvh.me |
| `Dockerfile` | workspace: `@ject-hub/service-frontend`, EXPOSE 5175 |

### 3. Очистка от хвостов Pulse

**Удалено:**
- `dist/` — старая сборка с pulse хвостами
- `src/Metrics.tsx` — не нужен для Service
- `src/Alerts.tsx` — не нужен для Service
- `src/PulseDashboard.tsx` — переименован в ServiceDashboard

**Обновлено:**
- `src/main.tsx` — ServiceLayout, menuItems с иконками bi bi-wrench/bi bi-grid/bi bi-gear
- `src/Dashboard.tsx` — re-export ServiceDashboard
- `src/ServiceDashboard.tsx` — новый компонент (создан)
- `src/pages/LoginPage.tsx` — комментарий "Service приложения"
- `src/pages/RegisterPage.tsx` — комментарий "Service приложения"
- `src/pages/DashboardPage.tsx` — переведён на русский, убраны pulse-специфичные данные
- `src/pages/DevicesPage.tsx` — комментарий "Service", UI на русском

### 4. Конфигурация vite.config.ts

**exposes:**
```typescript
const exposes = {
  './Dashboard': './src/Dashboard',
  './Devices': './src/Devices',
  './Settings': './src/Settings',
};
```

**manifest.json:**
- serviceId: 'service'
- name: 'Service'
- navigation: Dashboard (bi bi-wrench), Devices (bi bi-grid), Settings (bi bi-gear)
