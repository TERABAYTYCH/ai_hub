# 001.6: Devices — Загрузить устройства из Hub

## Цель

Убедиться, что Service frontend загружает устройства из Hub через HubProxyModule.

## Задачи

### 1. Проверить текущую реализацию

Убедиться что:
- `apps/service/frontend/src/Devices.tsx` (или `DevicesPage.tsx`) использует API для загрузки устройств
- API вызывает `GET /devices` на `api.service.lvh.me`
- HubProxyModule в service-backend проксирует запрос в Hub

### 2. Проверки

- [ ] Devices страница показывает устройства из Hub
- [ ] Устройства отображают имя, тип
- [ ] Ошибка если Hub недоступен

### 3. Проверки на отсутствие хвостов Pulse

- [ ] `grep -r "Pulse\|pulse" apps/service/frontend/src/Devices*` — 0 результатов
- [ ] API URL указывает на `api.service.lvh.me` (не pulse)
- [ ] Нет захардкоженных ссылок на pulse.lvh.me

## План

1. Проверить структуру файлов Devices в service-frontend
2. Проверить API URL
3. Запустить приложение и проверить загрузку устройств
4. Запустить grep проверки
