# 001.5: Hub Integration — Добавить Service в меню

## Цель

Добавить Service в Hub sidebar - зарегистрировать Service в `useMicroserviceManifests` чтобы Hub динамически загружал меню из manifest.json.

## Задачи

### 1. Изменить useMicroserviceManifests.ts

Добавить service в `REGISTERED_SERVICES`:

```typescript
export const REGISTERED_SERVICES = [
  { serviceId: 'pulse', manifestUrl: 'http://pulse.lvh.me/assets/manifest.json' },
  { serviceId: 'service', manifestUrl: 'http://service.lvh.me/assets/manifest.json' },
] as const;
```

### 2. Проверки

- [ ] Hub sidebar показывает Service в меню
- [ ] Клик на Service → раскрывает Dashboard, Devices, Settings
- [ ] Навигация работает: /service/dashboard, /service/devices, /service/settings

### 3. Проверки на отсутствие хвостов

- [ ] Нет дублирования serviceId в REGISTERED_SERVICES
- [ ] ServiceManifest корректен (содержит serviceId: "service")

## План

1. Прочитать useMicroserviceManifests.ts
2. Добавить service в REGISTERED_SERVICES
3. Запустить typecheck и lint
4. Проверить, что Service отображается в Hub меню
