# 001.3: Docker Compose — Добавить Service

## Цель

Добавить service-backend и service-frontend в docker-compose.yml.

## Задачи

### 1. Добавить в docker-compose.yml

**service-backend:**
- build: apps/service/backend
- container_name: ${COMPOSE_PROJECT_NAME}_service_backend
- ports: 3002:3002
- environment: NODE_ENV, PORT=3002, JWT_SECRET, HUB_INTERNAL_API_URL
- volumes: .:/app
- command: yarn workspace @ject-hub/service-backend start:dev
- depends_on: mysql-service, hub-backend

**service-frontend:**
- build: apps/service/frontend
- container_name: ${COMPOSE_PROJECT_NAME}_service_frontend
- ports: 5175:5175
- environment: NODE_ENV, VITE_API_URL=http://api.service.lvh.me, WATCHPACK_POLLING=true
- volumes: .:/app
- command: yarn workspace @ject-hub/service-frontend dev:docker
- depends_on: service-backend

### 2. Проверки

- [ ] `docker compose config` проходит без ошибок
- [ ] service-backend слушает порт 3002
- [ ] service-frontend слушает порт 5175

### 3. Проверки на отсутствие хвостов Pulse

- [ ] Нет упоминаний pulse в service-backend секции
- [ ] Нет упоминаний pulse в service-frontend секции
- [ ] service-backend не зависит от pulse-контейнеров
- [ ] service-frontend не зависит от pulse-контейнеров

## План

1. Прочитать текущий docker-compose.yml
2. Добавить service-backend секцию
3. Добавить service-frontend секцию
4. Запустить docker compose config
5. Запустить проверки
