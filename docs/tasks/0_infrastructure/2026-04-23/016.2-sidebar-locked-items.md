# Тикет 016.2: Infrastructure — Sidebar locked items

**Дата:** 2026-04-23  
**Домен:** 0_infrastructure  
**Родительский план:** PERMISSIONS_001 (Управление доступом к микросервисам)

---

## Цель

Добавить в Sidebar поддержку `locked` элементов меню — они должны показывать иконку замка вместо обычной иконки и быть визуально disabled.

---

## Текущее состояние

MenuItem не содержит полей `locked` и `disabled`.

---

## Требуемое состояние

### 1. Добавить в MenuItem:

```typescript
export interface MenuItem {
  id: string;
  label: string;
  icon?: string;
  path?: string;
  children?: MenuItem[];
  locked?: boolean;    // NEW
  disabled?: boolean;  // NEW
}
```

### 2. Изменить renderMenuItem в Sidebar:

Если `item.locked === true`:
- Использовать `bi bi-lock` вместо `item.icon`
- Применять стиль `opacity: 0.5; pointer-events: none;`
