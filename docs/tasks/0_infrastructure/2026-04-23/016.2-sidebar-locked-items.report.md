## Цель

Добавить в Sidebar поддержку `locked` элементов меню — иконка замка и визуально disabled стиль.

---

## Изменения

### Файл: `libs/ui-kit/src/components/layout/Sidebar.tsx`

**Обновлён MenuItem интерфейс:**
```typescript
export interface MenuItem {
  id: string;
  label: string;
  icon?: string;
  path?: string;
  children?: MenuItem[];
  locked?: boolean;    // NEW
  disabled?: boolean;   // NEW
}
```

**Обновлена логика рендеринга:**

- Если `locked === true`:
  - Показывается `bi bi-lock` вместо `item.icon`
  - Применяется стиль `opacity: 0.5; pointer-events: none`
  - Курсор `not-allowed`
  
- Дочерние элементы заблокированного родителя также disabled
