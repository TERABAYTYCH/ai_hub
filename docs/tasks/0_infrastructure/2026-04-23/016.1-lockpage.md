## Цель

Создать компонент LockPage в ui-kit для отображения "Модуль недоступен" при попытке зайти на заблокированный сервис.

---

## Текущее состояние

LockPage не существует.

---

## Требуемое состояние

Создать `libs/ui-kit/src/components/LockPage.tsx`:

```typescript
interface LockPageProps {
  serviceName?: string;
  returnUrl?: string;
}

export function LockPage({ serviceName = 'модуль', returnUrl = '/' }: LockPageProps) {
  return (
    <div className="d-flex flex-column align-items-center justify-content-center vh-100">
      <i className="bi bi-lock" style={{ fontSize: '5rem', color: '#6c757d' }}></i>
      <h2 className="mt-4">Модуль недоступен</h2>
      <p className="text-muted">
        Для активации {serviceName} обратитесь к администратору
      </p>
      <a href={returnUrl} className="btn btn-primary mt-3">
        На главную
      </a>
    </div>
  );
}
```

Экспортировать из `libs/ui-kit/src/index.ts`:
```typescript
export { LockPage } from './components/LockPage';
```
