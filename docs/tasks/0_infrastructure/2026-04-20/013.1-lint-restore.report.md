## Пример типизации API ответа

**Было (с `any`):**
```typescript
const response = await fetch(url);
const data = await response.json(); // any
```

**Стало (с `as`):**
```typescript
const response = await fetch(url);
const data = (await response.json()) as LoginResponseDto;
```

Использование DTO из `@app/contracts` для явного каста делает код типобезопасным без отключения правил.
