# Тикет 006.1: Hub → Federation host + manifest

**Дата:** 2026-04-22  
**Домен:** 2_pulse  
**Родительский тикет:** 006 (Pulse — добавить Service и Hub в навигацию)

---

## Цель

Превратить Hub из Federation consumer в Federation host, чтобы Pulse и Service могли загружать Hub Settings удалённо.

---

## Текущее состояние

- Hub `vite.config.ts` настроен как consumer с пустым remotes
- Нет exposes для ./Settings
- Нет manifestPlugin

---

## Требуемое состояние

Hub `vite.config.ts`:
```typescript
federation({
  name: 'hub',
  filename: 'remoteEntry.js',
  exposes: {
    './Settings': './src/pages/SettingsPage',
  },
  shared: ['react', 'react-dom', 'react-router-dom'],
})
```

+ manifestPlugin с moduleMapping для ./Settings

---

## Изменения

| Файл | Действие |
|------|----------|
| `apps/hub/frontend/vite.config.ts` | Изменить: убрать remotes, добавить exposes, добавить manifestPlugin |
