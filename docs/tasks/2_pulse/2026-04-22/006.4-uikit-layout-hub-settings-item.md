## Цель

Добавить проп `hubSettingsItem` в Layout для отображения Hub Settings в конце меню.

---

## Текущее состояние

Layout не имеет пропса для Hub Settings.

---

## Требуемое состояние

```typescript
export function Layout({
  children,
  serviceName = 'Ject Hub',
  staticMenuItems = [],
  excludeServices = [],
  hubSettingsItem,  // NEW
}: {
  children: ReactNode;
  serviceName?: string;
  staticMenuItems?: MenuItem[];
  excludeServices?: string[];
  hubSettingsItem?: MenuItem;  // NEW
}) {
  // ...
  // hubSettingsItem добавляется в конец menuItems
  const menuItems: MenuItem[] = [...staticMenuItems, ...serviceMenuItems, ...(hubSettingsItem ? [hubSettingsItem] : [])];
}
```
