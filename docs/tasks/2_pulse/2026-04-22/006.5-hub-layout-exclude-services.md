# Тикет 006.5: Hub → Layout с excludeServices

**Дата:** 2026-04-22  
**Домен:** 2_pulse  
**Родительский тикет:** 006 (Pulse — добавить Service и Hub в навигацию)

---

## Цель

Обновить Hub Layout для использования excludeServices, чтобы Hub не отображался в своём собственном меню.

---

## Текущее состояние

Hub Layout.tsx использует ui-kit Layout без excludeServices.

---

## Требуемое состояние

```typescript
// apps/hub/frontend/src/components/layout/Layout.tsx
import { Layout as UiKitLayout, type MenuItem } from '@ject-hub/ui-kit';

const hubMenuItems: MenuItem[] = [
  { id: 'hub-devices', label: 'Devices', icon: 'bi bi-grid', path: '/devices' },
  { id: 'hub-settings', label: 'Settings', icon: 'bi bi-gear', path: '/settings' },
];

export function Layout({ children }: { children: ReactNode }) {
  return (
    <UiKitLayout 
      serviceName="Ject Hub" 
      staticMenuItems={hubMenuItems}
      excludeServices={['hub']}  // NEW: exclude self
    >
      {children}
    </UiKitLayout>
  );
}
```

---

## Изменения

| Файл | Действие |
|------|----------|
| `apps/hub/frontend/src/components/layout/Layout.tsx` | Добавить excludeServices={['hub']} |
