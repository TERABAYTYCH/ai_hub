## Цель

Создать Pulse Layout с Hub Settings как самостоятельным пунктом меню.

## Изменения

### Файл: `apps/pulse/frontend/src/components/layout/Layout.tsx` (СОЗДАН)

```typescript
export function Layout({ children }: { children: ReactNode }) {
  return (
    <UiKitLayout
      serviceName="Pulse"
      staticMenuItems={pulseMenuItems}
      excludeServices={['pulse']}
      hubSettingsItem={{
        id: 'hub-settings',
        label: 'Hub Settings',
        icon: 'bi bi-gear',
        path: '/hub/settings',
      }}
    >
      {children}
    </UiKitLayout>
  );
}
```

### Файл: `apps/pulse/frontend/src/main.tsx` (ИЗМЕНЁН)

- Удалён локальный PulseLayout
- Импортирован Layout из `./components/layout/Layout`
- Все маршруты используют новый Layout

### Файл: `libs/ui-kit/src/components/layout/Layout.tsx` (ИЗМЕНЁН)

Добавлена логика авто-исключения Hub из manifests когда передан hubSettingsItem:

```typescript
const effectiveExcludeServices = hubSettingsItem
  ? [...excludeServices, 'hub']
  : excludeServices;
```

Это позволяет избежать дублирования Hub Settings — Hub не отображается как подпункт, только как单独的 hubSettingsItem.
