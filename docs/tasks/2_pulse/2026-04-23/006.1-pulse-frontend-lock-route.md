**Дата:** 2026-04-23

## Цель

Добавить route `/lock` в Pulse Frontend для отображения LockPage при попытке зайти на заблокированный сервис.

---

## Контекст

При прямом входе на `pulse.lvh.me/lock` пользователь должен увидеть LockPage. Это страница-заглушка для случаев, когда Pulse заблокирован.

**Примечание:** LockPage теперь без returnUrl пропса — Layout уже имеет навигацию.

---

## Требуемое состояние

### Добавить /lock route в Pulse App.tsx

**Файл:** `apps/pulse/frontend/src/App.tsx`

```typescript
import { LockPage } from '@ject-hub/ui-kit';

// в staticRoutes добавить:
{ path: '/lock', element: <LockPage serviceName="Pulse" /> }
```
