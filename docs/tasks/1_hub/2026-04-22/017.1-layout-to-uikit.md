# 017.1: Hub → ui-kit — Layout

## Цель

Скопировать Layout из Hub в ui-kit и адаптировать для универсального использования.

## Задачи

### 1. Изучить текущий Layout в Hub

Прочитать:
- `apps/hub/frontend/src/components/layout/Layout.tsx`
- `apps/hub/frontend/src/components/layout/Sidebar.tsx`
- `libs/ui-kit/src/components/layout/` (проверить что там есть)

### 2. Скопировать Layout в ui-kit

Создать в `libs/ui-kit/src/components/layout/`:
- `Layout.tsx` — адаптированный Layout
- `Sidebar.tsx` — адаптированный Sidebar (если нужен)
- `AppLayout.tsx` — alias/reexport для совместимости

### 3. Адаптация Layout

- Принимать `menuItems` пропсом (не хардкодить)
- Принимать `serviceName` пропсом
- Принимать `username` и `onLogout` пропсами
- Экспортировать типы `MenuItem`

### 4. Проверки

- [ ] Layout экспортируется из `@ject-hub/ui-kit`
- [ ] Hub продолжает работать как прежде (пока без изменений - это 017.2)

## План

1. Изучить Layout.tsx в Hub
2. Скопировать и адаптировать в ui-kit
3. Экспортировать из ui-kit/index.ts
4. Запустить typecheck
