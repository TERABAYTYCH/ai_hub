## Что сделано

Проверено наличие Layout в ui-kit.

### Текущая структура ui-kit/layout

ui-kit уже содержит все необходимые компоненты:

| Компонент | Файл | Описание |
|-----------|------|----------|
| `AppLayout` | `AppLayout.tsx` | Главный layout - обёртка с Header, Sidebar и контентом |
| `Sidebar` | `Sidebar.tsx` | Боковое меню с поддержкой вложенности |
| `Header` | `Header.tsx` | Шапка с именем сервиса и пользователем |
| `MenuItem` | `Sidebar.tsx` | Интерфейс элемента меню |

### Экспорты из ui-kit

```typescript
// libs/ui-kit/src/index.ts
export { Header, Sidebar, AppLayout, type MenuItem } from './components/layout';
```

### Hub Layout.tsx

Hub уже использует ui-kit:

```typescript
import { AppLayout, MenuItem, useAuth, useMicroserviceManifests } from '@ject-hub/ui-kit';

export function Layout({ children }) {
  // Строит menuItems из manifests
  return (
    <AppLayout menuItems={menuItems} serviceName="Ject Hub" username={username} onLogout={logout}>
      {children}
    </AppLayout>
  );
}
```
