# Тикет 017.1: Hub Backend — MICROSERVICES_ACCESS + JWT

**Дата:** 2026-04-23  
**Домен:** 1_hub  
**Родительский план:** PERMISSIONS_001 (Управление доступом к микросервисам)

---

## Цель

При логине пользователя добавить в JWT токен информацию о доступных микросервисах.

---

## Контекст

При логине пользователя создаётся JWT токен. В этот токен нужно добавить информацию о том, какие микросервисы доступны пользователю.

**Текущий JWT payload:**
```typescript
{
  sub: user.id,
  username: user.username,
  role: user.role,
  licenseId: user.licenseId ?? ''
}
```

**Нужно добавить:**
```typescript
{
  // ... existing fields
  microservices: {
    pulse: true,   // доступен
    service: false // заблокирован
  }
}
```

---

## Где искать

`apps/hub/backend/src/auth/auth.service.ts` — метод `generateTokens()` (строка ~83-100)

---

## Текущее состояние (generateTokens)

```typescript
private generateTokens(user: User): LoginResponseDto {
  const payload: UserJwtPayload = {
    sub: user.id,
    username: user.username,
    role: user.role,
    licenseId: user.licenseId ?? '',
  };
  // ...
}
```

---

## Требуемое состояние

### 1. Создать константу MICROSERVICES_ACCESS:

```typescript
const MICROSERVICES_ACCESS: Record<string, boolean> = {
  pulse: true,
  service: true,
};
```

### 2. Добавить microservices в payload:

```typescript
private generateTokens(user: User): LoginResponseDto {
  const payload: UserJwtPayload = {
    sub: user.id,
    username: user.username,
    role: user.role,
    licenseId: user.licenseId ?? '',
    microservices: MICROSERVICES_ACCESS, // NEW
  };
  // ...
}
```
