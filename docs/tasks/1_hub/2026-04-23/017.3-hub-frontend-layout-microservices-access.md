# Тикет 017.3: Hub Frontend — Layout передаёт microservicesAccess

**Дата:** 2026-04-23  
**Домен:** 1_hub  
**Родительский план:** PERMISSIONS_001 (Управление доступом к микросервисам)

---

## Цель

Добавить поле `microservices` в `IUser` интерфейс и передавать `microservicesAccess` из `useAuth()` в UiKitLayout.

---

## Контекст

Hub Layout в `apps/hub/frontend/src/components/layout/Layout.tsx` использует `UiKitLayout` из ui-kit. UiKitLayout уже поддерживает проп `microservicesAccess: Record<string, boolean>` (реализовано в 016.3).

**Решение:** Добавить `microservices` в `IUser` интерфейс. При логине `login()` уже получает accessToken — нужно его декодировать и извлечь `microservices` в payload.

---

## Требуемое состояние

### 1. Изменить IUser в AuthProvider

**Файл:** `libs/ui-kit/src/providers/AuthProvider.tsx`

Добавить поле `microservices` в `IUser`:

```typescript
export interface IUser {
  id: string;
  username: string;
  email?: string;
  firstName?: string;
  lastName?: string;
  microservices?: Record<string, boolean>; // <-- добавить
}
```

### 2. Изменить login в AuthProvider

При вызове `login(accessToken, user?)` — декодировать токен и извлекать `microservices`:

```typescript
const login = useCallback((accessToken: string, user?: IUser) => {
  setAccessToken(accessToken);
  
  // Декодируем токен для извлечения microservices
  const tokenPayload = decodeTokenPayload(accessToken);
  const microservices = (tokenPayload?.microservices as Record<string, boolean>) || undefined;
  
  setState({
    isAuthenticated: true,
    user: user ? { ...user, microservices } : null,
    isLoading: false,
  });
}, []);
```

### 3. Изменить Hub Layout

**Файл:** `apps/hub/frontend/src/components/layout/Layout.tsx`

```typescript
import { Layout as UiKitLayout, type MenuItem, useAuth } from '@ject-hub/ui-kit';

export function Layout({ children }: { children: ReactNode }) {
  const { user } = useAuth();
  const microservicesAccess = user?.microservices || {};

  return (
    <UiKitLayout
      serviceName="Ject Hub"
      staticMenuItems={hubMenuItems}
      excludeServices={['hub']}
      microservicesAccess={microservicesAccess}
    >
      {children}
    </UiKitLayout>
  );
}
```
