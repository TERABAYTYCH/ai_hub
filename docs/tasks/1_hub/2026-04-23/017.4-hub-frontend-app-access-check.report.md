## Доработки

**Доработка #1 - 2026-04-23 — Добавлена проверка isLoading**

При `user = null` (isLoading = true) `microservicesAccess` = `{}`, что приводило к некорректной проверке `isLocked`. Добавлена проверка `isLoading` в `useDynamicRoutesConfig` — если `isLoading = true`, возвращается пустой массив routes.

**Файлы:** `apps/hub/frontend/src/App.tsx`
