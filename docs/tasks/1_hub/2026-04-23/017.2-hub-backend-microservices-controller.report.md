## Цель

Создать API для управления доступом к микросервисам (Pulse, Service).

---

## Выполненные изменения

### 1. Создан MicroservicesController

**Файл:** `apps/hub/backend/src/microservices/microservices.controller.ts`

```typescript
@Controller('microservices')
export class MicroservicesController {
  @Get('access')
  getAccess() {
    return MICROSERVICES_ACCESS;
  }

  @Post('access')
  updateAccess(@Body() body: { pulse?: boolean; service?: boolean }) {
    if (body.pulse !== undefined) {
      MICROSERVICES_ACCESS.pulse = body.pulse;
    }
    if (body.service !== undefined) {
      MICROSERVICES_ACCESS.service = body.service;
    }
    return MICROSERVICES_ACCESS;
  }
}
```

### 2. Создан MicroservicesModule

**Файл:** `apps/hub/backend/src/microservices/microservices.module.ts`

```typescript
@Module({
  controllers: [MicroservicesController],
})
export class MicroservicesModule {}
```

### 3. Подключен MicroservicesModule в AppModule

**Файл:** `apps/hub/backend/src/app.module.ts`

Добавлен импорт `MicroservicesModule` в массив imports.
