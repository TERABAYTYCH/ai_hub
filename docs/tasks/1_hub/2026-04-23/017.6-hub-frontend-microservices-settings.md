## Цель

Создать страницу управления доступом к микросервисам (Pulse, Service). Администратор сможет включать/выключать доступ к модулям.

---

## Контекст

Администратору нужна страница для управления доступом к модулям. Она будет обращаться к `GET/POST /api/microservices/access` для получения и сохранения настроек.

---

## Требуемое состояние

### 1. Создать MicroservicesSettings.tsx

**Файл:** `apps/hub/frontend/src/pages/MicroservicesSettings.tsx`

```typescript
import { useState, useEffect } from 'react';
import { Form } from 'react-bootstrap';
import axios from 'axios';

interface MicroservicesAccess {
  pulse: boolean;
  service: boolean;
}

export default function MicroservicesSettings() {
  const [access, setAccess] = useState<MicroservicesAccess>({ pulse: true, service: true });
  const [saved, setSaved] = useState(false);

  useEffect(() => {
    axios.get('/api/microservices/access').then(res => setAccess(res.data));
  }, []);

  const handleToggle = (service: keyof MicroservicesAccess) => {
    setAccess(prev => ({ ...prev, [service]: !prev[service] }));
    setSaved(false);
  };

  const handleSave = async () => {
    await axios.post('/api/microservices/access', access);
    setSaved(true);
    setTimeout(() => setSaved(false), 2000);
  };

  return (
    <div className="container mt-4">
      <h2>Управление доступом к модулям</h2>
      <Form.Check
        type="switch"
        id="pulse-switch"
        label="Pulse"
        checked={access.pulse}
        onChange={() => handleToggle('pulse')}
      />
      <Form.Check
        type="switch"
        id="service-switch"
        label="Service"
        checked={access.service}
        onChange={() => handleToggle('service')}
      />
      <button className="btn btn-primary mt-3" onClick={handleSave}>Применить</button>
      {saved && <span className="ms-3 text-success">Сохранено!</span>}
    </div>
  );
}
```

### 2. Добавить route в App.tsx

**Файл:** `apps/hub/frontend/src/App.tsx`
