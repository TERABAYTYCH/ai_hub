# Тикет 017.2: Hub Backend — MicroservicesController API

**Дата:** 2026-04-23  
**Домен:** 1_hub  
**Родительский план:** PERMISSIONS_001 (Управление доступом к микросервисам)

---

## Цель

Создать API для управления доступом к микросервисам (Pulse, Service). Данные пока хранятся в памяти (позже — в БД).

---

## Контекст

Для управления доступом к микросервисам нужен API endpoint. Этот контроллер позволит администратору включать/выключать доступ к Pulse и Service в реальном времени.

---

## Требуемое состояние

### 1. Создать MicroservicesController

**Файл:** `apps/hub/backend/src/microservices/microservices.controller.ts`

```typescript
@Controller('microservices')
export class MicroservicesController {
  @Get('access')
  getAccess() {
    return MICROSERVICES_ACCESS;
  }

  @Post('access')
  updateAccess(@Body() body: { pulse?: boolean; service?: boolean }) {
    if (body.pulse !== undefined) {
      MICROSERVICES_ACCESS.pulse = body.pulse;
    }
    if (body.service !== undefined) {
      MICROSERVICES_ACCESS.service = body.service;
    }
    return MICROSERVICES_ACCESS;
  }
}
```

### 2. Создать MicroservicesModule

**Файл:** `apps/hub/backend/src/microservices/microservices.module.ts`

```typescript
@Module({
  controllers: [MicroservicesController],
})
export class MicroservicesModule {}
```

### 3. Импортировать в AppModule

**Файл:** `apps/hub/backend/src/app.module.ts`

Добавить `MicroservicesModule` в imports массив.
