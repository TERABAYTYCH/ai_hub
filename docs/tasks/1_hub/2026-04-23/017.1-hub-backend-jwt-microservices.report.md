# Отчёт 017.1: Hub Backend — MICROSERVICES_ACCESS + JWT

**Дата:** 2026-04-23  
**Домен:** 1_hub  
**Тип задачи:** PERMISSIONS_001 hub-backend  
**Статус:** ✅ Завершено

---

## Цель

При логине пользователя добавить в JWT токен информацию о доступных микросервисах.

---

## Выполненные изменения

### 1. Contracts — добавлено поле `microservices` в UserJwtPayload

**Файл:** `libs/contracts/src/hub/auth/index.ts`

```typescript
export interface UserJwtPayload {
  sub: string;
  username: string;
  role: string;
  licenseId: string;
  /** Доступность микросервисов { serviceId: boolean } */
  microservices: Record<string, boolean>;
  iat?: number;
  exp?: number;
}
```

### 2. AuthService — добавлена константа MICROSERVICES_ACCESS

**Файл:** `apps/hub/backend/src/auth/auth.service.ts`

```typescript
/**
 * Дефолтные значения доступа к микросервисам.
 * Позже будет браться из БД (таблица licenses).
 */
const MICROSERVICES_ACCESS: Record<string, boolean> = {
  pulse: true,
  service: true,
};
```

### 3. AuthService.generateTokens — включение microservices в payload

```typescript
private generateTokens(user: User): LoginResponseDto {
  const payload: UserJwtPayload = {
    sub: user.id,
    username: user.username,
    role: user.role,
    licenseId: user.licenseId ?? '',
    microservices: MICROSERVICES_ACCESS, // NEW
  };
  // ...
}
```
