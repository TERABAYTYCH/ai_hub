# Отчет 005.1-fix-device-api-auth

## Выполнено

Авторизация через JWT токен уже была реализована в `apps/hub/frontend/src/api/devices.api.ts`:

```typescript
function getAuthHeaders(): HeadersInit {
  const token = localStorage.getItem('accessToken');
  return {
    'Content-Type': 'application/json',
    ...(token ? { Authorization: `Bearer ${token}` } : {}),
  };
}
```

Все функции (`getDevices`, `getDevice`, `createDevice`, `updateDevice`, `deleteDevice`) используют `getAuthHeaders()` для добавления токена в заголовки.
