## Цель задачи

Исправить ошибки в созданном AppLayout: вернуть кнопку переключения темы, адаптировать Header и Sidebar под темную/светлую тему и починить отображение иконок в боковом меню.

---

## Выполненные работы

### 1. Добавлен пакет bootstrap-icons

**`apps/hub/frontend/package.json`:**
- Добавлена зависимость `bootstrap-icons@^1.13.1`
- Добавлен импорт CSS в `main.tsx`: `import 'bootstrap-icons/font/bootstrap-icons.css';`

### 2. Исправлен рендеринг иконок в Sidebar

**`libs/ui-kit/src/components/layout/Sidebar.tsx`:**
- Тип `icon` изменен с `React.ReactNode` на `string` (принимает классы like `bi bi-grid`)
- Иконка рендерится как `<i className={item.icon}></i>`
- Добавлен отступ `me-2` между иконкой и текстом

### 3. Возвращена кнопка переключения темы в Header

**`libs/ui-kit/src/components/layout/Header.tsx`:**
- Добавлен импорт `useTheme` из ThemeProvider
- Добавлена кнопка переключения темы (иконки bi-moon/bi-sun) рядом с профилем
- Кнопка использует `aria-label` для доступности
- Кнопки используют `variant="outline-secondary"` с CSS классами для адаптации под тему

### 4. Адаптация Header и Sidebar под темную тему

**Header:**
- Заменен `bg="primary"` на `var(--bs-body-tertiary)`
- Добавлен бордер `var(--bs-border-color)`
- Имя пользователя использует `var(--bs-body-color)`

**Sidebar:**
- Заменен `bg-light` на `var(--bs-body-bg)`
- Добавлен бордер справа `var(--bs-border-color)`
- Активный элемент использует `bg-primary text-white`
- Неактивный элемент использует `text-body`

### 5. Иконки вместо эмодзи в ThemeToggle

**`libs/ui-kit/src/theme/ThemeToggle.tsx`:**
- Заменены эмодзи ☀️/🌙 на иконки `bi bi-moon` / `bi bi-sun`

### 6. Исправление отображения имени пользователя

**`apps/hub/frontend/src/pages/RegisterPage.tsx`:**
- Добавлено сохранение `user` в localStorage после регистрации
- Добавлен автоматический редирект на /devices после успешной регистрации

**`apps/hub/frontend/src/App.tsx`:**
- Добавлена функция `getUsername()` для чтения имени из объекта `user` в localStorage
- Обновлен `handleLogout()` - удаляет `user` из localStorage

### 7. Добавлена страница Settings

**`apps/hub/frontend/src/pages/SettingsPage.tsx`:**
- Создана страница-заглушка для /settings

**`apps/hub/frontend/src/App.tsx`:**
- Добавлен маршрут `/settings`
- Settings добавлен в меню

### 8. CSS стили для кнопок Header

**`libs/ui-kit/src/components/layout/Header.css`:**
- Создан файл со стилями для кнопок
- `.header-btn` - цвет `var(--bs-body-color)`
- `.header-btn:hover` - белый цвет `#fff`
