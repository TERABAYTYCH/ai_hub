# Отчет 005.3-fix-login-redirect

## Выполнено

Обновлён `apps/hub/frontend/src/pages/LoginPage.tsx`:

1. **Редирект после успешного входа**: После сохранения токена вызывается `navigate('/devices')` вместо отображения сообщения об успехе.

2. **Редирект авторизованного пользователя**: Добавлен `useEffect` который проверяет наличие токена и редиректит на `/devices`, если пользователь уже авторизован.

```tsx
useEffect(() => {
  if (localStorage.getItem('accessToken')) {
    navigate('/devices');
  }
}, [navigate]);
```
